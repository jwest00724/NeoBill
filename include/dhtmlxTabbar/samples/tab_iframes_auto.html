TabBar - per page iframes

Set hrefmode attribute value to iframes and add href attributes to tab tags to load tab content in iframe, which will be autoimaticaly created for each tab.

<div hrefmode="ajax" id="a_tabbar" class="dhtmlxTabBar" imgpath="../imgs/" style="width:390; height:390;" skinColors="#FCFBFC,#F4F3EE" > <div id="b1" width="100" name="Tab 1" href="http://scbr.com"></div> <div id="b2" width="100" name="Tab 2" href="http://google.com"></div> <div id="b3" width="100" name="Tab 3" href="http://scand.com"></div> </div>
