DHTML JavaScript Tabbar samples

TabBar events

You can set event handler for tab selection. If it returns false then tab selection will not be changed.

<div id="a_tabbar" style="width:400px;height:100px"></div> <script language="javascript" type="text/javascript"> function my_func(idn, ido) { return (confirm("Tab "+idn+" to set active\nPlease Confirm?")); }; tabbar=new dhtmlXTabBar("a_tabbar","top"); tabbar.setImagePath("imgs/"); tabbar.setOnSelectHandler(my_func); tabbar.loadXML("tabs1.xml"); </script>
